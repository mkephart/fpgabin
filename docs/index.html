fpga.upload_to_fpga(REG_MEM=None, SEQ_MEM=None, PRG_MEM=None, HSK_SEL_MEM=None, VOLT_MEM=None)[source]¶

Writes binary files of appropriate type for each array loaded in. Iterables passed in must be of the correct order and size. If one of the types is not set, or is of the wrong size, it will be skipped (no values will be written).
